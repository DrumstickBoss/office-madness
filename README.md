## 二、作品創作理念

現代上班族每天都要在有限的時間與精力裡，同時應付好幾件事：一邊工作一邊接電話、一邊處理生活雜事一邊還要保持專業形象、一邊趕著完成任務一邊還得應付各種突發狀況。這種「一心多用、隨時被打斷」的日常，正是本作最初的靈感來源。

《社畜大亂鬥》以「社畜日常」為核心主題，將這些令人會心一笑又略帶心酸的職場情境，轉化成一系列輕鬆詼諧的雙重任務（Dual-Task）小遊戲：玩家一邊執行主要任務（接便當、丟垃圾等），一邊還要應付突如其來的干擾（推銷來電、麻煩的親友），藉此讓玩家在遊戲中體驗、也宣洩現實生活中的忙碌與荒謬感。

四款關卡分別對應上班族生活的不同切面——用餐、日常瑣事、理財、加班——彼此獨立卻共享同一套「一心多用 + 詼諧吐槽」的設計語言與美術世界觀，並以同一位主角「阿鬥」貫穿全場，讓玩家彷彿在扮演自己，會心一笑之餘也能獲得抒壓與共鳴。

## 三、世界觀與角色設定

- **主角「阿鬥」**：Q 版造型的平凡上班族吉祥物，是每位玩家在遊戲中的化身；名稱與造型刻意保持樸實、接地氣，強化「這就是你我」的既視感。
- **美術風格**：可愛、明亮的 Q 版美術，搭配帶點無厘頭、黑色幽默的文字吐槽（如職場垃圾話、詐騙電話台詞、親友吐槽等），形成「畫面萌、內容酸」的反差趣味。
- **遊戲大廳**：以一張城市地圖為主視覺，四個關卡以地圖上的據點呈現，玩家可自由選擇挑戰任一已開放關卡；大廳同時整合玩家等級／經驗值、排行榜與各關卡最高分紀錄，強化「持續回訪、刷新紀錄」的黏著度。

## 五、各關卡玩法說明

### 關卡一：🍱 社畜接便當

**情境**：午休時間，一邊接住掉落的便當果腹，一邊還得應付不斷打來的手機，稍不留神就會餓肚子又得罪人。

**基礎操作**

- 滑鼠移動 / 手指滑動：控制角色左右移動，平滑跟隨
- 接住 🟩 便當 → `+100 分`
- 碰到 💀 陷阱 → `−30 分`

**雙重任務：手機來電**

每隔 **10 秒**，畫面兩側會彈出來電，必須在 **3 秒內**做出正確反應：

| 來電類型                        | 正確動作   | 正確結果                      | 錯誤或逾時           |
| ------------------------------- | ---------- | ----------------------------- | -------------------- |
| 📞 正經電話（老闆、PM、主管）   | 點「接聽」 | 定身 3 秒，並顯示職場吐槽台詞 | 扣 50 分             |
| ⚠️ 詐騙電話（飆股、貸款、中獎） | 點「掛斷」 | 無懲罰，繼續接便當            | 分數歸零 + 定身 5 秒 |

**設計重點**：玩家必須在「顧著接便當」與「正確判斷來電性質」之間快速切換注意力，重現辦公室裡「一心多用還要不出錯」的緊繃感。

---

### 關卡二：♻️ 垃圾分類王

**情境**：面對源源不絕丟過來的垃圾，要眼明手快地滑動丟進正確的分類桶；途中還會有「不能被當垃圾丟掉」的家人朋友夾雜其中，考驗玩家的判斷力。

**遊戲流程**（共兩階段）

1. **丟垃圾階段（45 秒）**：手上同時持有 3 樣物品，玩家可自由選擇丟擲順序；用滑動（拖曳並放開）的方式將垃圾拋向畫面上的三個分類桶——資源回收 ♻️、廚餘 🐟、一般垃圾 🗑️。分類桶每隔一段時間會重新洗牌排列，逼迫玩家隨時重新判斷位置。
2. **Bonus Time 垃圾車加碼（20 秒）**：垃圾車進場後，物品改為由上方掉落，玩家操控接網左右移動接住物品得分。

**評分規則**

| 行為                | 結果      |
| ------------------- | --------- |
| 丟進正確分類桶      | `+100 分` |
| 丟進錯誤分類桶      | `−30 分`  |
| Bonus Time 接住物品 | `+50 分`  |

**特殊機制：「誘餌」角色**

手上的三樣物品中，有機率混入「迷路的老人家」或「查勤的女友」——這兩位**絕對不能被當成垃圾丟進分類桶**！畫面兩側會滑入對應的正確去處（老人家 → 輪椅／女友 → 計程車），玩家必須把他們丟向正確的目標：

| 行為                              | 結果                           |
| --------------------------------- | ------------------------------ |
| 丟進正確目標（如：老人家 → 輪椅） | `+100 分`                      |
| 誤丟進垃圾桶（把人當垃圾丟掉）    | `−500 分`（並有懲罰台詞吐槽）  |
| 丟錯目標（如：老人家 → 計程車）   | 未被丟中目標，也不算丟進垃圾桶 |
| Bonus Time 誤接到誘餌角色         | `−100 分`                      |

**設計重點**：以誇張荒謬的「差點把人當垃圾丟掉」情境製造笑點，同時測試玩家在時間壓力下能否保持判斷力，呼應日常生活中「趕時間卻還要顧全大局」的忙亂感。

### 關卡三：🏃 網銀運動會（開發中）

- **玩法**：角色會自動向前奔跑（橫向卷軸 Temple Run 跑酷模式），玩家需控制「跳躍（避開障礙物:水灘、欄杆）」，碰到跑鞋或是水瓶會變成無敵狀態(不怕任何障礙物)加速4秒鐘。
- **障礙**：隨時會有攝影師突然出現做拍攝(會面會閃白3秒)、啦啦隊揮舞旗子擋住畫面2秒，需點擊讓他們離開畫面。

### 關卡四：💼 加班修羅場（開發中）

- **主題**：以「加班日常」為發想方向，呈現下班前被追加工作、臨時會議、緊急需求輪番轟炸的處境。
- **視覺基調**：橘色系，延續全作 Q 版 + 詼諧吐槽的美術與敘事風格。
- **目前狀態**：關卡入口與世界觀定位已在大廳中保留，實際玩法機制規劃中，尚未開發。

## 六、成長與收藏系統

- **玩家等級／經驗值**：每完成一局即可獲得基礎經驗值，得分越高額外加成越多，等級曲線採「前期快、後期磨」的老派 RPG 成長節奏，鼓勵玩家持續遊玩。
- **星級評價**：每個已完成關卡依分數表現分為 ★☆☆／★★☆／★★★ 三個等級，讓玩家有明確的「再玩一次、拚更高分」動機。
- **排行榜**：各關卡各自累積前 10 名最高分紀錄（因各關卡計分尺度不同，排行榜不互相比較），激勵玩家反覆挑戰刷新紀錄。

### 環境需求

| 工具    | 版本                        |
| ------- | --------------------------- |

| npm     | `10.x`（建議 `10.5.0+`）    |

### 快速啟動

```bash
cd bento-blitz
npm install
npm run dev
```

開啟瀏覽器前往 → **https://drumstickboss.github.io/office-madness/**

### 其他指令

```bash
npm run build      # 編譯正式版本（輸出至 dist/）
npm run preview    # 預覽正式版本
npx tsc --noEmit   # TypeScript 型別檢查
```

### 技術選型

- **React + TypeScript**：元件化 UI + 型別安全
- **HTML5 Canvas**：高效能 2D 遊戲渲染
- **Vite**：極速 HMR 開發體驗
